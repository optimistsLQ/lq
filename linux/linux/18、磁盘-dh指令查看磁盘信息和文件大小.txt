在线上服务器下，一般是没有图形化界面的，经常有需求是查看当前磁盘的相关信息和使用情况，指令如下

1、df -l/-h/-H/-T/-t/-x   - 查看本地磁盘信息

相关选项：
-l   列出磁盘基本信息，以字节为单位   df -l
-h   以1024进制显示磁盘  df -l -h  或   df -lh
-H   以1000进制显示磁盘  df -l -H
-T   显示磁盘类型type  df -l -T
-t   搜索磁盘类型为xxx的磁盘  df -l -t ext4
-x   搜索除了xxx磁盘类型以外的磁盘  df -l -t ext4



2、du -b/-k/-m/-h/-H/-s 文件名/文件夹名   - 统计查看文件或文件夹大小

相关选项：
-b   - 以字节的形式显示文件信息
-k   - 以千字节的形式显示文件信息
-m   - 以兆的形式显示文件信息
-h   - 以1024进制形式显示文件信息
-H   - 以1000进制形式显示文件信息
-s   - 显示文件夹本来的大小
