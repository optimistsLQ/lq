位置变量： 在脚本代码中调用通过命令行传递给脚本的参数

如 hello.sh  -r -l   或  bash hello.sh a b c

如上命令的 -r 就是第一个参数 -l 就是第二个参数...


例： 
  bash hello.sh a b c ...

  echo 第一个参数是：$1;     - a
  echo 第二个参数是：$2;     - b
  echo 命令本身是：$0;      - hello.sh
  echo 所有的参数是：$*    - a b c 注意是一个字符串，不是三个
  echo 每一个参数：$@     - a b c 注意是三个字符串
  echo 参数个数：$#      - 3
  echo 清空一下所有的位置变量  
  set --
  echo 重新看一下所有变量：$@  



$1,$2,$...  - 对应第1，第2等参数
$0  - 命令本身
$*  - 所有的参数，全部参数合为一个字符串
$@  - 传递给脚本的所有参数，每个参数为独立字符串

注意： $* 和 $@只有被双引号包起来的时候才有差异， echo "$@"

$#  - 传递给脚本的参数的个数
Set--   - 清空所有的位置变量
