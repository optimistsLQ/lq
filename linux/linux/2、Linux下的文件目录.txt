/bin : 系统指令目录，存放系统指令，比如ls,mkdir指令等
/dev : 系统设备目录，硬盘、光驱都是在此目录下，如/dev/cdrom
/home : 用户目录，非超级管理员账号都在home目录下有对应的文件夹
/lib : 库文件目录，库一共分为两种： *.so后缀动态库文件， *.a后缀静态库文件
/media : 多媒体目录，光驱与软驱
/mnt : 挂载目录，如/mnt/udisk目录下
/proc : 内存目录，分配内存等
/sbin : 超级指令目录，s 是super的缩写，代表超级，给超级管理员使用的目录，如关机指令，删除某些文件指令
/tmp : temp缩写，代表临时目录
/var : 经常变化的目录，如启动文件、引导文件
/etc : 系统配置文件目录
/lost+found : 失物认领处，如系统崩溃时导致某些文件丢失
/opt : 第三方程序目录，如qq、wps等软件通常放置于此目录
/root : 超级管理员目录,与home目录类似
/usr : 相当于windows中的program files 目录，程序一般都安装在此目录
